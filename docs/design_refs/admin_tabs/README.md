## Style

- Warm off-white content background.
- Deep teal left sidebar.
- Bronze/gold accent borders and buttons.
- Quiet, trustworthy, cultural, tool-like admin UI.
- Dense but organized desktop browser layout.
- Cards use restrained 8px radius.
- Keep future action buttons visible for later implementation.
